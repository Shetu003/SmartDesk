      ℹ️ About

      SmartDesk is your intelligent productivity companion designed to empower individuals with tools for mindful working, task management, emotional well-being, and real-time activity tracking — all in one seamless web platform.

      Built using Spring Boot and integrated with Apache Kafka for real-time event streaming, SmartDesk ensures a responsive and interactive experience. It blends modern UI design with backend efficiency to help you stay on track and emotionally balanced throughout your day.

      Key Features:

      • Thought of the Day — Get inspired every morning with curated motivational quotes.
      • Focus Timer — Apply the Pomodoro technique to stay concentrated and reduce burnout.
      • Mood Journal — Track your emotions and receive uplifting responses based on your mood input.
      • Task Manager — Plan, track, and mark your daily tasks with ease and clarity.
      • Activity Feed — View real-time logs of your interactions and completed goals.

      Our Mission

      Our goal is to help individuals enhance productivity while staying mentally resilient. We believe that great work emerges from clarity, calmness, and emotional awareness — not just checklists.

      Technologies Used

      • Spring Boot – Backend RESTful APIs
      • Apache Kafka – Real-time event processing & logging
      • HTML, CSS, JavaScript – Frontend UI
      • MySQL – Persistent data storage
      • Docker & Docker Compose – Containerized deployment
